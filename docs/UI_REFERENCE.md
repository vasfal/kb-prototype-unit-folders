# UI Reference

## Figma (primary reference)

### Platform layout
**File:** https://www.figma.com/design/gpVOpetVBR1JleFug3U332/Layout-UI-improvements?node-id=1-108721

This file contains the current HRM platform layout: global sidebar, unit tree, top bar, tab bar.

### Knowledge Base design
**File:** https://www.figma.com/design/KDJ0aVVR1FmAt8ohgZyd3P/Knowledge-Base?node-id=24-880

This file contains the Knowledge Base module design. It defines:
- Card-based article grid layout (3-column grid, no folder tree panel)
- Folder sections as content headers with colored icons and article counts
- Article cards with title, unit name, status badge, time ago, and owner avatar
- Toolbar with search, filter, sort, sub-units toggle, grid/list view mode, create button
- Updated color palette (primary: #006bd6, active tab: #0052a3)
